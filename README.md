# Змейка++

Игра «Змейка» в браузере — HTML5/Canvas, чистый JavaScript, без зависимостей.
Автор: Лёша Карпич, 5-б.

## Запуск

Открыть `index.html` в любом браузере — и играть.

## Управление

| Клавиши | Действие |
|---------|----------|
| `←→↑↓` / `WASD` | Движение |
| `Пробел` | Пауза / снять паузу |
| `R` / `Enter` | Рестарт после проигрыша |
| `Q` / `Esc` | Выход в главное меню |
| `X` | Вызвать эффект салата (для теста) |

## Еда и механика

### 🔴 Яблоко (красный)
На поле всегда одно. Съел — +1 очко, змейка растёт на 1 клетку, скорость +1.

### 🟡 Картошка (жёлтый)
Появляется каждые 2–4 яблока, одна на поле. Съел — +2 очка, змейка растёт,
скорость падает до ⅔ на 3 секунды: можно успеть довернуть.

### 🔵 Пирог (голубой)
Появляется каждые 5–9 яблок и **не исчезает** — их может накопиться несколько.
Съел — **−5 очков**, змейка вырастает вдвое по длине. Осторожно!

### 🟣 Салат (фиолетовый)
Появляется каждые 9–13 яблок, один на поле. Съел — **+5 очков**, но змейка
укорачивается на ¼ длины. Хвостовые клетки исчезают одна за другой с анимацией.

## Правила

- Поле **сферическое**: выйдя за край, змейка появляется с противоположной стороны
- Врежься в **себя** — конец игры
- Скорость растёт от 1 до 50 по мере поедания еды
- На паузе поле становится чёрно-белым
