# Snake

Простая игра «Змейка» на Python + Pygame.

## Запуск

```bash
pip install pygame
python main.py
```

## Управление

| Клавиши | Действие |
|---------|----------|
| `←→↑↓` / `WASD` | Движение |
| `R` | Рестарт после проигрыша |
| `Esc` / `Q` | Выход |

## Структура проекта

```
snake/
├── main.py           # Точка входа
├── src/              # Игровая логика (будущие модули)
├── assets/
│   ├── images/       # Спрайты и текстуры
│   ├── sounds/       # Звуковые эффекты и музыка
│   └── fonts/        # Шрифты
├── .gitignore
└── README.md
```

## Правила

- Съедай красную еду — змейка растёт, счёт увеличивается
- Скорость растёт каждые 5 очков
- Столкновение со стеной или собой — конец игры
