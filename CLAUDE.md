# Змейка++ — CLAUDE.md

Игра «Змейка++» на HTML5/Canvas/JavaScript. Автор: Лёша Карпич, 5-б.

## Запуск

Открыть `index.html` в браузере — никаких зависимостей не нужно.

## Структура файлов

```
snake-web/
  index.html   — весь игровой код (HTML + JS в одном файле)
  VERSION      — текущая версия
  CLAUDE.md    — этот файл
```

## Архитектура главного цикла

Цикл работает через `requestAnimationFrame`. Движение змейки управляется
таймером (`lastTickMs`), что позволяет анимациям быть плавными независимо
от игровой скорости. Скорость плавно интерполируется: `speed` → `targetSpeed`
с коэффициентом `SPEED_LERP` ед/с.

## Поле

Сферическое (тороидальное): правый край соединён с левым, верх — с низом.
Игра заканчивается только при столкновении змейки с самой собой.

## Игровые объекты

| Объект | Цвет | Условие появления | Очки | Эффект |
|--------|------|-------------------|------|--------|
| Яблоко (красный) | #dc2828 | всегда 1 на поле | +1 | рост на 1, скорость +1 (0.2 ticks/s) |
| Картошка (жёлтый) | #ffbe00 | каждые randInt(2,4) яблока, 1 на поле | +2 | рост на 1, скорость ×⅔ на 3с |
| Пирог (голубой) | #00c8ff | каждые randInt(5,9) яблок, накапливается | −5 | рост до 2× текущей длины |
| Салат (фиолетовый) | #a032dc | каждые randInt(9,13) яблок, 1 на поле | +5 | укорочение на ¼ длины |

## Механика роста/укорочения

**growPending** (пирог): `growPending += preLen - 1`. На каждом обычном тике
если `growPending > 0`: хвост не удаляется, `growPending--`. Змейка растёт
с `preLen` до `2 × preLen`.

**shrinkPending** (салат): на каждом обычном тике если `shrinkPending > 0`:
дополнительно удаляется клетка с текущего конца хвоста с анимацией
(вспышка белым → зелёный → прозрачность), `shrinkPending--`.

## Анимация исчезания хвоста (cellAnims)

Массив `cellAnims` — каждый элемент `{ pos: [x,y], start: ms }`.
Длительность одной клетки `CELL_DUR = 350мс`:
1. 0–60мс: белая вспышка
2. 60–140мс: белый → зелёный
3. 140–350мс: зелёный → прозрачный

## Спаун объектов

- Картошка: `applesEaten == nextOrangeAt`, +randInt(2,4) после каждого спауна
- Пирог: `applesEaten == nextCakeAt`, +randInt(5,9), накапливается в `cakes[]`
- Салат: `applesEaten == nextSaladAt`, +randInt(9,13), одна на поле

## Скорость

Отображается как целое число 1–50. Формула:
`dispSpd(s) = round(1 + (s - SPEED_INIT) / SPEED_STEP)`, где
`SPEED_STEP = 0.2` (1 ед = 0.2 ticks/s), `SPEED_MAX = 12.3` (display 50).

## Пауза

**ПРОБЕЛ** переключает паузу. На паузе поле обесцвечивается, надпись «ПАУЗА»
пульсирует: `br = 130 + 125 × (sin(t × π × 1.5) + 1) / 2`.

## Анимация статуса

3-фазная (вспышка → дрожание → угасание), `STATUS_DUR = 2500мс`.

| Событие | Текст | Цвет |
|---------|-------|------|
| Яблоко | Восторг! | красный |
| Картошка | Картошечка! | золотой |
| Пирог | Ой! Я съела нельзЮ.. | голубой |
| Салат | Разгрузочный день! | фиолетовый |

## Ключевые константы

| Константа | Значение | Описание |
|-----------|----------|----------|
| `SPEED_INIT` | 2.5 | Начальная скорость (ticks/s) |
| `SPEED_MAX` | 12.3 | Максимальная скорость (display 50) |
| `SPEED_STEP` | 0.2 | ticks/s на одну единицу отображаемой скорости |
| `SPEED_LERP` | 3 | Скорость интерполяции (ед/с) |
| `ORANGE_DUR` | 3000 мс | Длительность эффекта картошки |
| `STATUS_DUR` | 2500 мс | Длительность анимации статуса |
| `CELL_DUR` | 350 мс | Длительность анимации одной клетки хвоста |

## Версионирование

Версия хранится в `VERSION` и в константе `VERSION` внутри `index.html`.
При обновлении менять оба места.
