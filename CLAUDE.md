# Змейка — CLAUDE.md

Игра «Змейка» на HTML5/Canvas/JavaScript. Автор: Лёша Карпич, 5-б.

## Запуск

Открыть `index.html` в браузере — никаких зависимостей не нужно.

## Структура файлов

```
snake-web/
  index.html   — весь игровой код (HTML + JS в одном файле)
  VERSION      — текущая версия
  CLAUDE.md    — этот файл
```

## Архитектура главного цикла

Цикл работает через `requestAnimationFrame`. Движение змейки управляется
таймером (`lastTickMs`), что позволяет анимациям быть плавными независимо
от игровой скорости.

## Игровые объекты

| Объект | Цвет | Условие появления | Эффект |
|--------|------|-------------------|--------|
| Яблоко (красный) | #dc2828 | всегда 1 на поле | +1 очко, рост, +0.3 к скорости |
| Апельсин (жёлтый) | #ffbe00 | каждые randInt(2,4) яблока, 1 на поле | +1 очко, рост, скорость ×⅔ на 3с |
| Пирог (голубой) | #00c8ff | каждые randInt(5,9) яблок, накапливается | рост до 2× текущей длины (очков не даёт) |

### Механика пирога (growPending)
При съедании пирога: `growPending += preLen - 1` (где `preLen` — длина до вставки головы).
На каждом обычном тике (без еды) если `growPending > 0`: хвост не удаляется,
`growPending -= 1`. Итого змейка вырастает с `preLen` до `2 × preLen`.

### Спаун объектов
- Апельсин: `applesEaten == nextOrangeAt`, стартует в `randInt(2,4)`,
  после каждого спауна сдвигается на `+randInt(2,4)` — всегда 1 на поле
- Пирог: `applesEaten == nextCakeAt`, стартует в `randInt(5,9)`,
  сдвигается на `+randInt(5,9)` — добавляется в массив `cakes[]`, не исчезает
- Если оба триггера совпали: апельсин спаунится первым, пирог избегает его позицию

## Пауза

**ПРОБЕЛ** переключает паузу. На паузе:
- Игровой тик не выполняется
- Таймер эффекта апельсина не тикает
- Поле рисуется обесцвеченным (фильтр `grayscale` через `ctx.filter`)
- Надпись «ПАУЗА» пульсирует: `brightness = 130 + 125 × (sin(t × π × 1.5) + 1) / 2`

При снятии паузы: `lastTickMs = now` — сброс таймера тика.

## Анимация статуса

`makeStatusSurf(elapsedMs, text, baseColor)` — 3-фазная анимация:
1. **Вспышка** (0–0.5с): рост 40→100%, тёмный → белый
2. **Дрожание** (0.5–1.5с): насыщенный ↔ белый (~2.5 Гц)
3. **Угасание** (1.5–2.5с): белый → тёмный, alpha 255→0

| Событие | Текст | Цвет |
|---------|-------|------|
| Яблоко | Восторг! | красный |
| Апельсин | Перевариваю! | золотой |
| Пирог | Ой! Я съела нельзя.. | голубой |

## Ключевые константы

| Константа | Значение | Описание |
|-----------|----------|----------|
| `SPEED_INIT` | 2.5 | Начальная скорость (шаг/с) |
| `SPEED_MAX` | 22 | Максимальная скорость |
| `ORANGE_DUR` | 3000 мс | Длительность эффекта апельсина |
| `STATUS_DUR` | 2500 мс | Длительность анимации статуса |
