# Змейка++ — CLAUDE.md

Игра «Змейка++» на HTML5/Canvas/JavaScript. Автор: Лёша Карпич, 5-б.

## Запуск

Открыть `index.html` в браузере — никаких зависимостей не нужно.

## Структура файлов

```
snake-web/
  index.html   — весь игровой код (HTML + JS в одном файле)
  VERSION      — текущая версия (сейчас 2.07)
  CLAUDE.md    — этот файл
  README.md    — описание для GitHub
```

## Состояния игры

`state` принимает значения: `'start'` → `'playing'` → `'entry'` → `'gameover'` → `'start'`.

- **start** — стартовый экран с правилами и таблицей рекордов
- **playing** — игра идёт
- **entry** — ввод имени после смерти (показан `#nameBox`)
- **gameover** — экран результата с таблицей рекордов

## Архитектура главного цикла

Цикл работает через `requestAnimationFrame`. Движение змейки управляется
таймером (`lastTickMs`), что позволяет анимациям быть плавными независимо
от игровой скорости. Скорость плавно интерполируется: `speed` → `targetSpeed`
с коэффициентом `SPEED_LERP` ед/с.

## Поле

Сферическое (тороидальное): правый край соединён с левым, верх — с низом.
Игра заканчивается только при столкновении змейки с самой собой.

## Механика роста/укорочения

**growPending** (пирог): `growPending += preLen - 1`. На каждом обычном тике
если `growPending > 0`: хвост не удаляется, `growPending--`. Змейка растёт
с `preLen` до `2 × preLen`.

**shrinkPending** (салат): на каждом обычном тике если `shrinkPending > 0`:
дополнительно удаляется клетка с текущего конца хвоста с анимацией
(вспышка белым → зелёный → прозрачность), `shrinkPending--`.

## Анимация исчезания хвоста (cellAnims)

Массив `cellAnims` — каждый элемент `{ pos: [x,y], start: ms }`.
Длительность одной клетки `CELL_DUR = 350мс`:
1. 0–60мс: белая вспышка
2. 60–140мс: белый → зелёный
3. 140–350мс: зелёный → прозрачный

## Спаун объектов

- Картошка: `applesEaten == nextOrangeAt`, +randInt(2,4) после каждого спауна
- Пирог: `applesEaten == nextCakeAt`, +randInt(5,9), накапливается в `cakes[]`
- Салат: `applesEaten == nextSaladAt`, +randInt(9,13), одна на поле

## Скорость

Отображается как целое число 1–50. Формула:
`dispSpd(s) = round(1 + (s - SPEED_INIT) / SPEED_STEP)`, где
`SPEED_STEP = 0.2` (1 ед = 0.2 ticks/s), `SPEED_MAX = 12.3` (display 50).

## Пауза

**ПРОБЕЛ** переключает паузу. На паузе поле обесцвечивается, надпись «ПАУЗА»
пульсирует: `br = 130 + 125 × (sin(t × π × 1.5) + 1) / 2`.

## Анимация статуса

3-фазная (вспышка → дрожание → угасание), `STATUS_DUR = 2500мс`.

| Событие | Текст | Цвет |
|---------|-------|------|
| Яблоко | Восторг! | красный |
| Картошка | Картошечка! | золотой |
| Пирог | Ой! Я съела нельзЮ.. | голубой |
| Салат | Разгрузочный день! | фиолетовый |

## Мобильная поддержка

D-pad (`#dpad`) — сетка 3×3 из HTML-кнопок под канвасом. Кнопки 110×96px,
обрабатывают `touchstart` с `preventDefault()` чтобы не триггерить скролл.

Масштабирование под экран телефона: `fitToScreen()` ставит `zoom = min(1, innerWidth/600)`
на `#gameWrap`. CSS `zoom` сжимает и layout, и visual size — канвас + D-pad
масштабируются вместе. `getBoundingClientRect()` возвращает зумированные координаты,
поэтому множитель `(WIN_W / rect.width)` корректно переводит clientX → canvas X.

## Выбор языка

Константы кнопок RU/EN на стартовом экране:

```js
const LB_W = 120, LB_H = 46, LB_GAP = 16;
const LB_RU_X  = WIN_W / 2 - LB_W - LB_GAP / 2;
const LB_EN_X  = WIN_W / 2 + LB_GAP / 2;
const _BLK_H   = 52 + 12 + LB_H + 12 + 1 + 14 + 240 + 14 + 1 + 12 + 24; // 428
const LB_BTN_Y = Math.round((WIN_H - _BLK_H) / 2) + 52 + 12;
```

`_BLK_H` — суммарная высота центрированного блока стартового экрана. Если меняется
любая составляющая (LB_H, отступы), `_BLK_H` нужно пересчитать, иначе hit-test сломается.

Нажатие обрабатывается через `handleLangTap(clientX, clientY)`, которую вызывают
и `click` (десктоп), и `touchend` (мобила). `touchend` использует `e.changedTouches[0]`
и `preventDefault()` чтобы не дублировать с синтетическим click.

## Таблица рекордов (leaderboard)

Хранится в JSONBin.io. Константы в начале скрипта:

```js
const BIN_ID  = '6a25dd4bda38895dfe968750';
const BIN_KEY = '$2a$10$...';
const BIN_URL = `https://api.jsonbin.io/v3/b/${BIN_ID}`;
```

`lbStatus`: `'loading'` | `'ok'` | `'error'`. Запись добавляется после окончания игры
(state → 'entry'), имя вводится в `#nameBox`. Если имя пустое — запись не отправляется.
